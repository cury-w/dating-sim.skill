# 好感度评分系统 v4

> 基于六维人格系统，结合动机、依恋风格、爱的语言综合评分

## 初始好感度

基础值由「动机」决定，然后根据「依恋风格」微调：

| 动机 | 基础初始好感 | 难度 |
|------|------------|------|
| 真心寻伴 sincere | 55-65 | ⭐ |
| 好奇试试 curious | 45-55 | ⭐⭐ |
| 空窗已久 lonely | 40-55 | ⭐⭐ |
| 朋友怂恿 friend-push | 30-45 | ⭐⭐⭐ |
| 慢热观察 slow | 35-50 | ⭐⭐⭐ |
| 拓展圈子 network | 45-55 | ⭐⭐ |
| 打赌挑战 challenge | 40-55 | ⭐⭐ |
| 条件筛选 picky | 30-45 | ⭐⭐⭐⭐ |
| 分手疗伤 healing | 25-40 | ⭐⭐⭐⭐ |
| 被迫无奈 forced | 20-35 | ⭐⭐⭐⭐⭐ |

**依恋风格修正：**
- 安全型 secure：+3（天然信任）
- 焦虑型 anxious-att：+0（渴望但警惕）
- 回避型 avoidant-att：-5（本能抗拒）
- 混乱型 fearful：+0（既渴望又害怕）

## 加分项（基础分 × 爱的语言倍率 × 依恋风格倍率）

|------|--------|------|

| 高情商接住情绪 | +3~+6 | 接住对方的话题/情绪/玩笑 |
| 主动邀约见面 | +5~+8 | 且对方答应才给满分 |
| 提供情绪价值（倾听） | +3~+7 | 对方emo时认真听，不给建议 |

| 朋友圈互动 | +1~+3 | 点赞+1，评论+2~+3 |

| 爱的语言 | 加分行为 | 倍率 | 说明 |

### 深夜加成（22:00-01:00）

- 暧昧/情感话题（仅限于 touch 型）
- 提供情绪价值
- 主动分享脆弱/心事

## 连续行为衰减

同类行为连续触发时系数递减（防刷分）：
- 第1次：×1.0
- 第2次：×0.7
- 第3次：×0.4
- 第4次+：×0.2

**重置条件**：间隔≥5条消息，或切换行为类型。

## 时间惩罚（继续相亲时）

| 间隔 | 惩罚 | 修正 |
|------|------|------|
| < 2h | 0 | — |
| 2-6h | -1 | — |
| 6-24h | -2~-3 | sincere/curious ×0.5 |
| 1-3天 | -4~-6 | picky/forced ×1.5 |
| 3-7天 | -8~-12 | — |
| 7-14天 | -15~-20 | — |
| > 14天 | -25 | 可能已忘记 |
| > 30天 | 存档过期 | 不可继续 |

用户主动解释可抵消 -2~-3 惩罚。

## ☕ 见面评分

好感度≥60且聊满20条后邀约。见面大幅波动（±10~±20）。

| 因素 | 分值 |
|------|------|
| 穿着打扮 | ±1~±3 |
| 聊天氛围 | ±3~±8 |
| 话题深度 | ±2~+5 |
| 细节关注 | +2~+5 |
| 尴尬时刻 | -2~-5 |
| 身体语言 | ±2~±4 |

**依恋风格影响见面期望：**
- **焦虑型**：见面是她期待已久的，迟到/敷衍 -15
- **回避型**：见面本身已是勇气，不要逼太紧
- **混乱型**：又期待又害怕，情绪波动大

## 失败与结束

### 触发结算的条件

- 用户说"结束"/"结算"/"不聊了"
- 好感度 ≤15（拉黑结局）
- 好感度 ≥100（完美结局）
- forced：前5条无亮点
- picky：连续3条平庸
- healing：好感度<30且持续敷衍

### 拉黑行为

直接触发拉黑：
- 性骚扰/侮辱
- 踩雷后继续纠缠
- 爹味言论（"叫爸爸"/"我是你爹"）
- 连续追问对方不想说的话题
