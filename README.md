# 💕 Dating Time — AI 相亲恋爱模拟器 v3

> 在微信里谈恋爱，但对方是 AI。

一个基于 [OpenClaw](https://github.com/openclaw/openclaw) Skill 框架的相亲恋爱模拟器。每次匹配一个拥有独立性格、职业、兴趣、雷区的虚拟对象，从加微信好友开始，一路聊到确定关系（或者拉黑）。

## ✨ 特性

### 🎲 115万种性格组合

6 个维度随机组合，每个对象都是独一无二的人：

| 维度 | 选项数 | 示例 |
|------|--------|------|
| 性格底色 | 18种 | 温暖治愈、犀利敏锐、元气满满、焦虑紧绷、叛逆不羁… |
| 社交能量 | 5种 | 深度社恐、内敛慢热、社交活跃… |
| 来相亲的动机 | 10种 | 真心寻伴、好奇试试、分手疗伤、被迫无奈… |
| 表达方式 | 8种 | 直球坦诚、傲娇别扭、毒舌犀利、梗王本王… |
| 依恋风格 | 4种 | 安全型、焦虑型、回避型、混乱型 |
| 爱的语言 | 5种 | 言语肯定、高质量陪伴、收到礼物、服务行动、身体接触 |

**理论组合数：18 × 5 × 10 × 8 × 4 × 5 = 1,152,000 种**

### 🧠 依恋风格系统

了解对方的依恋风格 = 知道什么行为会加分，什么行为会扣分：
- **安全型**：正常聊天就好，稳定加分
- **焦虑型**：给足安全感，及时回应，不要玩消失
- **回避型**：给空间，不要追太紧，她反而会慢慢靠近
- **混乱型**：保持稳定，不被她的反复带节奏

### 💬 爱的语言系统

说什么话最能打动她？根据对方的「爱的语言」决定：
- **言语肯定**：真诚具体的赞美效果翻倍
- **高质量陪伴**：认真倾听，人在心在
- **收到礼物**：记住纪念日和小惊喜
- **服务行动**：记住她说过的小事，用行动表达
- **身体接触**：暧昧/想念/想见面的话加分翻倍

### 📱 真实微信聊天模拟

- **微信主页** — 加好友后展示微信号、地区、个性签名，模拟真实第一感
- **朋友圈系统** — 加好友后先翻朋友圈，3-5条动态暗示性格和生活状态
- **分段发送** — 真实的人会一条消息分成2-4条发出来（根据性格决定频率）
- **「正在输入...」** — 对方在想、在组织措辞时的真实等待感
- **撤回消息** — 真实的人会发完后悔，1-3次/局
- **打错字/纠正** — 偶尔用，不要用太多
- **语音消息** — 特定性格会发语音（开车/情绪激动/比打字舒服）
- **回复节奏** — 对方不是秒回机器。在忙/开车/洗澡时自然延迟
- **已读不回** — 连续敷衍或踩雷后，对方真的会不回复你
- **困意收尾** — 对方想结束时会给台阶下，不要继续追话题
- **依恋风格影响行为** — 焦虑型会主动追问（不是踩雷！），回避型会后退

对方是真人，不是客服：
- **敷衍检测** — 连续回"嗯""哦""好吧"，对方从冷淡 → 无语 → 直接不回
- **踩雷真的会生气** — 不是每次都原谅你
- **依恋风格决定反应** — 焦虑型会追问，回避型会后退，混乱型矛盾纠结
- **冒犯就拉黑** — 爹味发言、性骚扰、连续冒犯 → 直接拉黑

### 💾 存档与继续

- **暂停相亲** — 保存进度，下次继续
- **继续相亲** — 列出所有存档，选择对象恢复对话
- **时间惩罚** — 3天没聊好感度-8~-12，7天-15~-20，30天过期
- 长时间不聊后对方会问"你最近消失了吗"

## 🎮 玩法

### 快速开始

对 AI 说：

```
开始相亲
```

填写你的个人信息（姓名/年龄/职业/性格/兴趣/期望），系统会自动匹配一个对象。

### 游戏流程

```
填档案 → 匹配对象 → 看朋友圈 → 微信聊天 → （邀约见面）→ 确定关系 / 拉黑
```

### ⚠️ 重要规则

- **匹配后不可更换对象**。匹配完成就锁定，不提供"换一个"选项
- 依恋风格决定对方行为：焦虑型追问不是踩雷，回避型后退不要追太紧
- 爱的语言决定加分点：说对话加很多，说错话扣分更深

### 常用命令

| 命令 | 效果 |
|------|------|
| 开始相亲 | 新开一局 |
| 暂停相亲 | 保存进度 |
| 继续相亲 | 选择存档继续 |
| 结束相亲 / 结算 | 查看结算报告 |

### 结局

| 结局 | 好感度 |
|------|--------|
| 🚫 拉黑 | ≤15 |
| 🙅 没聊起来 | <30 |
| 🤝 朋友 | 30-55 |
| 💕 互有好感 | 55-75 |
| 😏 暧昧 | 75-90 |
| ❤️ 在一起 | ≥90 |
